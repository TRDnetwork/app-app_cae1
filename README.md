This app is a static personal portfolio site with no database requirements. It includes a hero section, about paragraph, responsive project grid, and contact form. All functionality is client-side or handled via Vercel serverless functions — specifically, the contact form is processed by an external `api/contact.ts` function using Resend for email delivery. No Supabase tables, storage, auth, or edge functions are used. The frontend uses React + Vite + Tailwind with Framer Motion for scroll animations and adheres to warm minimalism design principles.
